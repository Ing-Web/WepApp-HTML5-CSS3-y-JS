Un menú vertical y horizontal con FLEX

Para crear un menú, debemos usar una lista desordenada, en los li creamos los hiperbinculos. La propiedad flex-direction con los valores row y column nos permite espeficicar si el vertical u horizontal

Tambien debemos aplicar el modificador list-style-type:none; para quitar las viñetas y realizar otras modificaciones para lograr el efecto deseado
