Página personal de Hernán Velásquez

Descripción de la actividad

Crear una página web personal simulada, que incluya al menos:

  1. Uso de colores para texto y fondo, incluyendo al menos colores degradados en alguna de sus cajas.
  2. Inclusión de tipografías externas de Google Fonts, Adobe Edge Web Fonts o un servicio similar on line.
  3. Distintos encabezados.
  4. Uso de caracteres especiales
  5. Incluir, imágenes de tipo bitmap y vectorial.
  6. Incluir algún vídeo.
  7. Incluir bordes redondeados en alguna de sus cajas.
  8. La página debe comenzar con el siguiente encabezamiento: "Página personal de <nombre y apellido del alumno>"

1. Uso de colores para texto y fondo, incluyendo al menos colores degradados en alguna de sus cajas

Para cumplir con este requirimiento, he usado dos colores principales, el marrón, cuyo código hexadecimal es #6d4c41; y el azul, con código #64b5f6; También he usado un degradado para las etiquetas <section>, se trata de un degradado lineal que usa los mismos dos colores, marrón y azul, también una sombra al lado derecho para lograr que se note los límites y no se confunda con el fondo de la etiqueta <html>.

2. Inclusión de tipografías externas de Google Fonts, Adobe Edge Web Fonts o un servicio similar on line

En este caso, estoy haciendo uso del servicio de Google, que cuenta con una amplia variedad de fuentes, para este ejemplo, estoy usando la fuente Ubuntu en todo el documento, excepto para mostrar los códigos de los colores y las etiquetas de HTML5, para los cuales estoy usando la fuente Raleway

3. Distintos encabezados.

Se han usado encabezados <h1>, <h2> y <h3> de diferentes tamaños, además se les ha aplicado las pseudo clases hover y active. Se han usado diferentes tamaños y los mismos colores para cada uno de los encabezados

4. Uso de caracteres especiales

En las anteriores secciones se han usado los caracteres especiales < y > para encerrar las etiquetas HTML. ahora vamos a escribir algunas fórmulas matemáticas para incluir otros caracteres especiales:

  • ∫ dx = x+C → Integral de x
  • L = (2*π*r*a)/360o → Longitud de un arco de circunferencia
  • A=π*r2 → Área del círculo

7. Incluir bordes redondeados en alguna de sus cajas.

Se han incluido muchos bordes redondeados, la etiqueta <body> tiene un leve borde redondeado en las cuatro esquinas. Las etiquetas <section> poseen bordes redondeados diferentes en las esquinas superiores y en las inferiores.
